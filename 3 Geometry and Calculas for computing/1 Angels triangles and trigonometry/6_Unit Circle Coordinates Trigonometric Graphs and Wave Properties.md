# Unit Circle Coordinates, Trigonometric Graphs, and Wave Properties

---

# 1. Coordinates on the Unit Circle

## What is the Unit Circle?

The **Unit Circle** is a special circle:

- Centered at the origin (0,0)
- Radius = 1

```text
          (0,1)
             *
          *     *
       *           *
(-1,0)*     O       *(1,0)
       *           *
          *     *
             *
          (0,-1)
```

Where:

```text
O = Origin (0,0)
```

and

```text
Radius = 1
```

---

# Point Moving Around the Circle

Imagine a point P moving around the edge of the circle.

```text
           P(x,y)
             *
            /|
           / |
          /  |
         /θ  |
        O----*
```

The angle from the positive x-axis is:

```text
θ
```

---

# Relationship with Trigonometry

For any point P:

```text
x = cos(θ)
```

```text
y = sin(θ)
```

Therefore:

```text
P = (cos(θ), sin(θ))
```

---

# Why Does This Work?

Since the radius is:

```text
1
```

SOH-CAH-TOA becomes:

```text
sin(θ)
=
Opposite/1
=
Opposite
```

```text
cos(θ)
=
Adjacent/1
=
Adjacent
```

So:

- Horizontal distance = cosine
- Vertical distance = sine

---

# Example 1

At:

```text
θ = 0°
```

```text
cos(0°)=1
sin(0°)=0
```

Coordinates:

```text
(1,0)
```

---

# Example 2

At:

```text
θ = 90°
```

```text
cos(90°)=0
sin(90°)=1
```

Coordinates:

```text
(0,1)
```

---

# Example 3

At:

```text
θ = 180°
```

```text
cos(180°)=-1
sin(180°)=0
```

Coordinates:

```text
(-1,0)
```

---

# Example 4

At:

```text
θ = 270°
```

```text
cos(270°)=0
sin(270°)=-1
```

Coordinates:

```text
(0,-1)
```

---

|---------|---------|
| 0° | (1,0) |

# 2. Derivation and Behavior of the Sine Graph

## What Does the Sine Graph Show?

The sine graph shows how the vertical height changes as a point moves around the unit circle.

Remember:

```text
sin(θ)
=
Vertical Height
```

---

# Starting Point

At:

```text
θ = 0
```

the point is:

```text
(1,0)
```

Height:

```text
0
```

Therefore:

```text
sin(0)=0
```

---

# At π/2 (90°)

The point reaches the top.

```text
(0,1)
```

Height:

```text
1
```

Therefore:

```text
sin(π/2)=1
```

Maximum value.

---

# At π (180°)

The point reaches the left side.

```text
(-1,0)
```

Height:

```text
0
```

Therefore:

```text
sin(π)=0
```

---

# At 3π/2 (270°)

The point reaches the bottom.

```text
(0,-1)
```

Height:

```text
-1
```

Therefore:

```text
sin(3π/2)=-1
```

Minimum value.

---

# At 2π (360°)

The point returns to start.

```text
(1,0)
```

Height:

```text
0
```

Therefore:

```text
sin(2π)=0
```

---

# Sine Graph Shape

```text
 1 |       *
   |     *   *
   |    *     *
 0 |***         *** 
   |               *
   |                *
-1 |                  *
   +--------------------
     0 π/2 π 3π/2 2π
```

---

# Important Sine Values

| Angle | sin(θ) |
|---------|---------|
| 0 | 0 |
| π/2 | 1 |
| π | 0 |
| 3π/2 | -1 |
| 2π | 0 |

---

# Real-Life Example

### Sound Waves

Microphones record sound as combinations of sine waves.

---

### Ocean Waves

The rise and fall resembles sine-wave motion.

---

### Electricity

AC electricity follows a sine-wave pattern.

# 4. Behavior and Asymptotes of the Tangent Graph

## What is Tangent?

Tangent measures slope (gradient).

```text
tan(θ)
=
Opposite/Adjacent
```

or

```text
tan(θ)
=
sin(θ)/cos(θ)
```

---

# Gradient Meaning

Gradient means:

```text
Rise
----
Run
```

---

# At θ = 0

Line is flat.

```text
tan(0)=0
```

---

# At θ = π/4 (45°)

Rise equals run.

```text
tan(π/4)=1
```

---

# Approaching π/2

As the line becomes vertical:

```text
Adjacent → 0
```

Since:

```text
tan(θ)
=
Opposite/Adjacent
```

division becomes enormous.

Therefore:

```text
tan(θ) → +∞
```

---

# Just Past π/2

The value suddenly flips:

```text
tan(θ) → -∞
```

---

# This Creates an Asymptote

An asymptote is a line that a graph approaches but never touches.

For tangent:

```text
x = π/2
```

is a vertical asymptote.

---

# Other Asymptotes

```text
x = 3π/2
```

```text
x = 5π/2
```

```text
x = 7π/2
```

and so on.

---

# Tangent Graph Shape

```text
      /
     /
    /
---|---
   /
  /
 /
```

Repeated forever.

---

# Key Tangent Values

| Angle | tan(θ) |
|---------|---------|
| 0 | 0 |
| π/4 | 1 |
| π/2 | Undefined |
| π | 0 |
| 5π/4 | 1 |

---

# Real-Life Example

### Road Slope

Steeper roads have larger tangent values.

---

### Roof Design

Tangent helps measure roof steepness.

# Final Key Idea

The **Unit Circle** connects geometry and trigonometry:

```text
Point = (cos(θ), sin(θ))
```

As a point moves around the circle:

- **Sine** tracks vertical height.
- **Cosine** tracks horizontal distance.
- **Tangent** tracks slope (gradient).

Plotting these values creates the famous trigonometric graphs:

- Sine Wave
- Cosine Wave
- Tangent Curve

These graphs are fundamental in mathematics, physics, engineering, computer graphics, signal processing, communications, machine learning, sound systems, and many real-world technologies.
