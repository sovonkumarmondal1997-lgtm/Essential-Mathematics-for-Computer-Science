# Mapping Functions to Graphs, Gradients, Linear Equations, and Connecting Algebra to Geometry

---

# Introduction

One of the most important ideas in mathematics, engineering, physics, and computer science is learning how to move between:

```text
Algebra
```

and

```text
Geometry
```

Algebra gives us equations such as:

\[
y = 2x + 1
\]

while geometry allows us to visualize these equations as shapes and lines on a graph.

Graphing is the bridge between these two worlds.

---

# 1. Mapping Functions to the Cartesian Plane

## What is the Cartesian Plane?

The Cartesian Plane (Coordinate Plane) is a flat surface used to plot points and graphs.

It contains two perpendicular axes:

```text
Vertical Axis   = y-axis
Horizontal Axis = x-axis
```

---

## Visual Representation

```text
          y
          ↑
          |
          |
          |
----------+----------→ x
          |
          |
          |
```

---

# Axis Definitions

## x-axis

The horizontal axis.

Represents:

```text
Inputs
```

or

```text
Domain
```

---

## y-axis

The vertical axis.

Represents:

```text
Outputs
```

or

```text
Range
```

---

# What Does Graphing Mean?

Graphing means:

```text
Convert an equation into points.
```

Then:

```text
Plot the points.
```

Finally:

```text
Connect the points.
```

to reveal the shape of the function.

---

# Example 1

Given:

\[
y=x+1
\]

Choose some x-values.

| x | y=x+1 |
|---|---|
| 0 | 1 |
| 1 | 2 |
| 2 | 3 |
| 3 | 4 |

---

# Plot the Points

```text
(0,1)
(1,2)
(2,3)
(3,4)
```

---

# Graph

```text
4 |       *
3 |     *
2 |   *
1 | *
0 +-------------
    0 1 2 3
```

The points form a straight line.

---

# Example 2

Given:

\[
y=x^2
\]

Calculate values.

| x | y |
|---|---|
| -2 | 4 |
| -1 | 1 |
| 0 | 0 |
| 1 | 1 |
| 2 | 4 |

---

# Graph Shape

```text
4 | *       *
3 |
2 |
1 |   *   *
0 |     *
  +-------------
   -2 -1 0 1 2
```

This creates a curve called a parabola.

---

# Real-Life Example

Suppose:

```text
x = Number of hours worked
```

and

```text
y = Total earnings
```

Graphing helps visualize how earnings change over time.

---

# 2. Geometric Interpretation of Gradient

## What is Gradient?

Gradient measures:

```text
Steepness
```

of a line.

Another name is:

```text
Slope
```

---

# Formula

\[
\text{Gradient}
=
\frac{\text{Rise}}{\text{Run}}
\]

or

\[
m
=
\frac{\Delta y}{\Delta x}
\]

---

# Meaning

Rise:

```text
Vertical change
```

Run:

```text
Horizontal change
```

---

# Example 1

Suppose a line goes:

```text
Up 4 units
Across 2 units
```

Gradient:

\[
m=\frac{4}{2}
\]

\[
m=2
\]

---

# Visualization

```text
      *
     /
    /
   /
  *
```

Steep upward line.

---

# Example 2

Suppose:

```text
Up 2 units
Across 4 units
```

Gradient:

\[
m=\frac{2}{4}
\]

\[
m=\frac12
\]

Much less steep.

---

# Trigonometric Connection

Recall:

\[
\tan(\theta)
=
\frac{\text{Opposite}}{\text{Adjacent}}
\]

This is exactly:

```text
Rise / Run
```

Therefore:

\[
\text{Gradient}
=
\tan(\theta)
\]

---

# Example: 45° Line

For:

\[
\theta=45^\circ
\]

we know:

\[
\tan(45^\circ)=1
\]

Therefore:

```text
Gradient = 1
```

---

# Visualization

```text
     *
    /
   /
  /
 *
```

The line rises exactly as much as it moves across.

---

# Real-Life Example

Road signs often display slope information.

```text
10% Gradient
```

means:

```text
Road rises 10 units
for every 100 units traveled.
```

---

# 3. The Linear Equation Framework (y = mx + c)

## What is a Linear Equation?

A linear equation creates a straight line.

General form:

\[
y = mx + c
\]

---

# Meaning of m

\[
m
\]

represents:

```text
Gradient (Slope)
```

---

# Meaning of c

\[
c
\]

represents:

```text
Y-Intercept
```

---

# What is the Y-Intercept?

The y-intercept is where the line crosses:

```text
y-axis
```

---

# How to Find It?

Set:

\[
x=0
\]

---

# Example

Given:

\[
y=2x+3
\]

Set:

\[
x=0
\]

\[
y=3
\]

Therefore:

```text
Y-intercept = (0,3)
```

---

# Example 2

Given:

\[
y=-4x+5
\]

Set:

\[
x=0
\]

\[
y=5
\]

Therefore:

```text
Y-intercept = (0,5)
```

---

# Understanding the Slope

Given:

\[
y=2x+3
\]

The slope is:

```text
2
```

Meaning:

```text
Up 2
Across 1
```

---

# Example Table

| x | y |
|---|---|
| 0 | 3 |
| 1 | 5 |
| 2 | 7 |
| 3 | 9 |

Notice:

```text
Every time x increases by 1,
y increases by 2.
```

---

# Real-Life Example

Taxi Fare:

\[
\text{Fare}
=
50 + 10x
\]

where:

```text
50 = starting charge
10 = charge per kilometer
```

This follows:

\[
y=mx+c
\]

---

# 4. Visualizing Positive vs. Negative Slopes

The sign of the gradient completely changes the direction of a line.

---

# Positive Gradient

If:

\[
m>0
\]

the line rises.

---

# Example

\[
y=2x-3
\]

Slope:

```text
+2
```

Y-intercept:

```text
-3
```

---

# Graph Shape

```text
      *
     /
    /
   /
  *
```

Moving left to right:

```text
Line goes upward.
```

---

# Example Values

| x | y |
|---|---|
| 0 | -3 |
| 1 | -1 |
| 2 | 1 |
| 3 | 3 |

---

# Real-Life Example

Savings account growth.

As time increases:

```text
Money increases.
```

Positive slope.

---

# Negative Gradient

If:

\[
m<0
\]

the line falls.

---

# Example

\[
y=-\frac12 x+3
\]

Slope:

\[
-\frac12
\]

Y-intercept:

```text
3
```

---

# Graph Shape

```text
 *
  \
   \
    \
     *
```

Moving left to right:

```text
Line goes downward.
```

---

# Example Values

| x | y |
|---|---|
| 0 | 3 |
| 2 | 2 |
| 4 | 1 |
| 6 | 0 |

---

# Real-Life Example

Fuel remaining in a car.

As distance increases:

```text
Fuel decreases.
```

Negative slope.

---

# Comparison

| Gradient | Behavior |
|-----------|-----------|
| Positive | Rising line |
| Negative | Falling line |
| Zero | Flat line |

---

# Example of Zero Gradient

\[
y=4
\]

Graph:

```text
4 | ***********
3 |
2 |
1 |
0 +------------
```

Flat horizontal line.

# 5. Bridging the Algebraic and Geometric Domain

## Why Is This Important?

A major goal in mathematics is to instantly connect:

```text
Equation
```

with

```text
Picture
```

---

# Algebra

Given:

\[
y=2x+1
\]

---

# Geometry

You should immediately know:

```text
Y-intercept = 1
Slope = 2
```

and imagine:

```text
An upward-sloping line.
```

---

# Finding Two Points

A straight line can be drawn using only:

```text
Two points
```

---

# Example

Given:

\[
y=3x-2
\]

---

## Point 1

Set:

\[
x=0
\]

\[
y=-2
\]

Point:

```text
(0,-2)
```

---

## Point 2

Set:

\[
x=1
\]

\[
y=1
\]

Point:

```text
(1,1)
```

---

# Draw the Line

Connect:

```text
(0,-2)
```

and

```text
(1,1)
```

You now have the complete graph.

---

# Quick Mental Interpretation

When you see:

\[
y=5x-7
\]

you can instantly read:

```text
Slope = 5
```

```text
Crosses y-axis at -7
```

No need to calculate many points.

# Final Key Idea

Graphing functions is the process of transforming:

```text
Algebra → Geometry
```

A linear equation:

\[
y = mx + c
\]

contains all the information needed to draw a straight line:

- **m** tells you how steep the line is.
- **c** tells you where it crosses the y-axis.

By understanding gradients, intercepts, and coordinates, you can quickly move between equations and visual graphs—a foundational skill in mathematics, physics, engineering, computer science, data science, and machine learning.
