# Rational Numbers

## Definition

A **Rational Number** is any number that can be written as:

```text
p/q
```

where:

```text
p and q are integers

q ≠ 0
```

### Symbol

```text
Q
```

---

## Examples

### Example 1

```text
1/2
```

---

### Example 2

```text
3/4
```

---

### Example 3

```text
5

=
5/1
```

---

### Example 4

```text
-7/3
```

---

## Decimal Examples

All of these are rational:

```text
0.5

0.75

2.25

3.125
```

because they can be converted into fractions.

# Irrational Numbers

## Definition

An **Irrational Number** cannot be written exactly as:

```text
p/q
```

where:

```text
p and q are integers.
```

Its decimal expansion:

```text
Never ends

Never repeats
```

---

## Examples

### Example 1

```text
π

=
3.141592653589793...
```

---

### Example 2

```text
√2

=
1.41421356237...
```

---

### Example 3

```text
√3

=
1.73205080756...
```

---

### Example 4

```text
e

=
2.718281828...
```

# Mersenne Primes

## Definition

A **Mersenne Prime** has the form:

```text
2^p - 1
```

where:

```text
p is a prime number.
```

---

## Example 1

```text
2³ - 1

=
7
```

Prime.

Binary:

```text
111₂
```

---

## Example 2

```text
2⁵ - 1

=
31
```

Prime.

Binary:

```text
11111₂
```

---

## Example 3

```text
2⁷ - 1

=
127
```

Prime.

Binary:

```text
1111111₂
```

---

## Pattern

Mersenne primes always look like:

```text
111

11111

1111111
```

in binary.

# Algebraic Conversion of Repeating Decimals to Fractions

## Definition

Convert repeating decimals into fractions using algebra.

---

## Example 1

Convert:

```text
0.333...
```

Let:

```text
x = 0.333...
```

Multiply by 10:

```text
10x = 3.333...
```

Subtract:

```text
10x - x

=
3.333... - 0.333...

=
3
```

```text
9x = 3
```

```text
x = 3/9
```

```text
x = 1/3
```

---

## Example 2

Convert:

```text
0.666...
```

Let:

```text
x = 0.666...
```

Multiply by 10:

```text
10x = 6.666...
```

Subtract:

```text
9x = 6
```

```text
x = 6/9
```

```text
x = 2/3
```

---

## Example 3

Convert:

```text
0.777...
```

Let:

```text
x = 0.777...
```

Multiply by 10:

```text
10x = 7.777...
```

Subtract:

```text
9x = 7
```

```text
x = 7/9
```

# Loop Termination Criteria

## Definition

In programming, mathematical properties can be used to decide when a loop should stop.

---

## Example 1

Prime Number Checking

```python
for i in range(2,n):
    if n % i == 0:
        stop
```

If divisor found:

```text
Number is not prime.
```

Loop stops.

---

## Example 2

Finding Factors

```python
while factor_found == False:
    continue
```

Stop when factor is found.

---

## Example 3

Searching for Prime Numbers

```python
keep checking

until divisor appears
```

Then terminate.

# Computer Number Representation and System Limits

## Definition

Computers cannot store every real number exactly.

They have limited memory and finite bits.

---

# Example 1: Irrational Numbers

```text
π

=
3.141592653589793...
```

Computer stores:

```text
3.141592653589793
```

only up to a certain precision.

Not exact.

---

## Example 2: √2

Actual:

```text
1.414213562373095048...
```

Computer:

```text
1.4142135623730951
```

Approximation.

---

## Example 3: Huge Numbers

A computer with:

```text
32-bit integer
```

can store only up to:

```text
2,147,483,647
```

Larger numbers require more storage.

---

# Memory Tricks

```text
Prime → Exactly two divisors.
```

```text
Composite → More than two divisors.
```

```text
Rational → Can be written as p/q.
```

```text
Irrational → Cannot be written as p/q.
```

```text
Repeating Decimal → Rational.
```

```text
Non-Repeating Infinite Decimal → Irrational.
```

```text
Real = Rational + Irrational.
```
