# Proof by Contrapositive - Beginner Friendly Notes

# 1. Proof by Contrapositive

## What is Proof by Contrapositive?

A proof by contrapositive is an **indirect proof** technique used to
prove a statement of the form:

If A, then B

Instead of proving:

A → B

directly, we prove its logically equivalent statement:

¬B → ¬A

If the contrapositive is true, then the original statement is also true.

Think of it as proving the same fact from the opposite direction.

------------------------------------------------------------------------

## Why use it?

Sometimes proving A → B directly is difficult.

The opposite direction is often much easier.

------------------------------------------------------------------------

## Example 1

Claim:

If a number is divisible by 4, then it is even.

Direct statement:

Divisible by 4 → Even

Contrapositive:

Not Even → Not Divisible by 4

If a number is odd,

it cannot be divisible by 4.

Therefore,

the original statement is true.

------------------------------------------------------------------------

## Example 2

Claim:

If a student gets full marks,

then the student passed.

Contrapositive:

If the student did not pass,

then the student did not get full marks.

Much easier to verify.

------------------------------------------------------------------------

# 2. Conditional Equivalence Foundation

The reason proof by contrapositive works is because

A → B

and

¬B → ¬A

always have exactly the same truth values.

They are logically equivalent.

Formula:

A → B ≡ ¬B → ¬A

------------------------------------------------------------------------

## Example

Original

If it rains,

roads become wet.

Contrapositive

If roads are not wet,

then it did not rain.

Both statements always agree.

If one is true,

the other is also true.

------------------------------------------------------------------------

# 3. Conditional Implication Testing Rule

Remember the truth table for implication.

An implication is false **only** in one situation:

Premise = True

Conclusion = False

Every other case is True.

Truth Table

  A   B   A→B
  --- --- -----
  T   T   T
  T   F   F
  F   T   T
  F   F   T

### Example

"If I study, then I pass."

The only failure is:

I study

but

I do not pass.

Every other situation satisfies the implication.

------------------------------------------------------------------------

# 4. Parity Property Verification

A common application of proof by contrapositive involves even and odd
integers.

## Example

Claim:

If n³ is even,

then n is even.

Instead of proving directly,

prove the contrapositive.

Contrapositive:

If n is odd,

then n³ is odd.

------------------------------------------------------------------------

### Step 1

Assume n is odd.

Definition of odd:

n = 2k + 1

------------------------------------------------------------------------

### Step 2

Cube both sides.

n³ = (2k + 1)³

Expand:

= 8k³ + 12k² + 6k + 1

Factor:

= 2(4k³ + 6k² + 3k) + 1

This has the form:

2m + 1

Therefore,

n³ is odd.

The contrapositive is proved.

Hence,

If n³ is even,

then n is even.

------------------------------------------------------------------------

# 5. Inequality Transformation

Proof by contrapositive is also useful with inequalities.

## Example

Claim:

If

x² \> y²

then

x \> y

(This statement is not true for all real numbers, but it is true under
additional conditions such as x and y being nonnegative. The algebraic
transformation below illustrates the technique.)

Instead of proving directly,

rewrite the difference.

Suppose

y \> x

Then

y − x \> 0

Now examine

y² − x²

Factor:

y² − x²

=

(y − x)(y + x)

Since

y − x \> 0

and (under the nonnegative assumption)

y + x \> 0

their product is positive.

Therefore,

y² \> x².

This proves the contrapositive under the stated conditions.

------------------------------------------------------------------------

# Real-Life Examples

## Programming

If a program crashes,

then some error occurred.

Contrapositive:

If no error occurred,

then the program did not crash.

------------------------------------------------------------------------

## Security

If a user logs in,

the password was correct.

Contrapositive:

If the password was incorrect,

the user did not log in.

------------------------------------------------------------------------

## Mathematics

Many proofs involving divisibility, parity, and inequalities are much
easier using contrapositives.

# Memory Tricks

-   Contrapositive = Reverse and negate.
-   A → B is equivalent to ¬B → ¬A.
-   Implication fails only when True → False.
-   Odd numbers have the form 2k + 1.
-   Rewrite inequalities using positive differences when helpful.
-   Use contrapositive whenever it is easier than a direct proof.

Proof by contrapositive is widely used in number theory, discrete
mathematics, algorithms, formal verification, compiler correctness, and
theoretical computer science.
