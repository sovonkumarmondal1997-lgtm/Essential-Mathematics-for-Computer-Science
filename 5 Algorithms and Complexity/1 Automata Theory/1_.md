# Formal Languages Basics (Simple Explanation)

## 1. Alphabet (`\Sigma`{=tex})

**Definition:**\
An **alphabet** is a **non-empty set of symbols** used to build strings.

Think of an alphabet like a **box of letters or symbols** you are
allowed to use.

### Example 1

Let

\[ `\Sigma`{=tex}={a,b,c} \]

Available symbols are:

-   a
-   b
-   c

You cannot use `d` because it is not part of the alphabet.

### Real-life Example

English alphabet:

    {A, B, C, ..., Z}

Binary alphabet:

    Σ = {0,1}

------------------------------------------------------------------------

## 2. String (or Word)

**Definition:**\
A **string** is a **finite sequence of symbols** taken from an alphabet.

Think of a string as a **word made from allowed symbols**.

### Example

If

    Σ = {a,b}

Possible strings are:

    a
    b
    ab
    ba
    aab
    abba
    babaa

Not valid:

    ac

because `c` is not in the alphabet.

------------------------------------------------------------------------

## 3. Empty String (ε)

**Definition:**\
The **empty string**, written **ε (epsilon)**, is a string containing
**zero symbols**.

It is **not a space** and **not the character "0"**.

### Example

For

    Σ = {a,b}

Examples:

    ε
    a
    ab
    bbb

The empty string contains nothing.

Length:

    |ε| = 0

Think of it as an empty word.

------------------------------------------------------------------------

## 4. Length of a String

**Definition:**\
The **length** of a string is the **number of symbols** it contains.

Notation:

    |x|

### Examples

String:

    apple

Length:

    |apple| = 5

Binary string:

    10101

Length:

    |10101| = 5

Another example:

    x = abba

    |abba| = 4

Empty string:

    |ε| = 0

------------------------------------------------------------------------

## 5. Sigma Star (Σ\*)

**Definition:**\
Σ\* is the set of **all possible strings**, including the empty string.

Think:

> Any length is allowed (0, 1, 2, 3, ...).

### Example

If

    Σ = {a,b}

Then

    Σ* =
    {
    ε,
    a,
    b,
    aa,
    ab,
    ba,
    bb,
    aaa,
    aab,
    aba,
    ...
    }

Contains:

-   empty string
-   one-letter strings
-   two-letter strings
-   infinitely many longer strings

------------------------------------------------------------------------

## 6. Sigma Plus (Σ⁺)

**Definition:**\
Σ⁺ is the set of **all non-empty strings**.

Difference from Σ\*:

-   Σ\* includes ε.
-   Σ⁺ does **not** include ε.

### Example

If

    Σ = {a,b}

Then

    Σ⁺ =
    {
    a,
    b,
    aa,
    ab,
    ba,
    bb,
    aaa,
    ...
    }

Notice:

    ε ∉ Σ⁺

------------------------------------------------------------------------

## 7. Sigma K (Σᵏ)

**Definition:**\
Σᵏ is the set of all strings having **exactly length k**.

### Example

Let

    Σ = {0,1}

### Σ¹

    {0,1}

### Σ²

    {
    00,
    01,
    10,
    11
    }

### Σ³

    {
    000,
    001,
    010,
    011,
    100,
    101,
    110,
    111
    }

Every string has exactly 3 symbols.

------------------------------------------------------------------------

## 8. Language

**Definition:**\
A **language** is simply a **collection (set) of strings** chosen from
an alphabet.

A language does **not** need to contain every possible string.

### Example

Alphabet:

    Σ = {a,b}

Language:

    L = {ab, aab, abb}

These strings belong to the language.

The following do not:

    aa
    bbb
    ε

unless explicitly included.

Think of Σ\* as the **entire dictionary** and a language as **selected
words from that dictionary**.

------------------------------------------------------------------------

## 9. Symbols vs. Strings of Length One

This is a common point of confusion.

### Symbol

A symbol is a single element of the alphabet.

Example:

    Σ = {a,b}

Here

    a

is a symbol.

### String of Length One

A string of length one is a string containing exactly one symbol.

Examples:

    "a"
    "b"

These belong to Σ¹.

### Analogy

Imagine Scrabble.

Individual tile:

    A

is a symbol.

Putting one tile on the table:

    "A"

is a string of length one.

### Comparison

  Item   Meaning
  ------ ---------------------------
  Σ      Set of symbols
  a      One symbol
  "a"    A string of length one
  Σ¹     All strings of length one
  Σ\*    All strings (including ε)
  Σ⁺     All non-empty strings

------------------------------------------------------------------------

# Quick Summary

  -----------------------------------------------------------------------
  Concept            Simple Meaning                    Example
  ------------------ --------------------------------- ------------------
  Alphabet (Σ)       Allowed symbols                   {a,b}

  String             Sequence of symbols               abba

  Empty String (ε)   String with no symbols            ε

  Length             Number of symbols                 \|abba\| = 4

  Σ\*                All strings including ε           ε, a, ab, abb, ...

  Σ⁺                 All non-empty strings             a, b, ab, abb, ...

  Σᵏ                 Strings of exactly length k       Σ²={aa,ab,ba,bb}

  Language           Selected strings from Σ\*         {ab,aab,abb}

  Symbol vs String   Symbol is one character; string   a vs "a"
                     is a sequence (even of length 1)
